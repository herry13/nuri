Nuri
====

Dynamic Workflow Configuration Tool.


Ubuntu
------
Required Packages
- ruby (>= 1.8.7)
- rubygems
- libz-dev
- libxml2-dev
- libaugeas-ruby

Required Ruby Gems
- mongrel
- json
- libxml-ruby
- antlr3

Required Packages

- rubygems
- zlib-devel
- libxml2-devel
- augeas-devel

Required Ruby Gems
- mongrel
- json
- libxml-ruby
- antlr3
