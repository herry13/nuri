Nuri The Automated Workflow Configuration Tool
"Define your desired state and let Nuri achieves it for you"

Why use Nuri

During the process of a configuration change, the order of the actions must be carefully chosen to maintain certain global constraints throughout the entire transition. Nuri is a configuration tool which can automatically generate and execute a workflow to move from one (declarative) configuration state to another while respecting any such constraints.
Some tools allow you to define such workflow manually. However, even in a small system, a very large number of workflows can be required to cater for every eventuality. And choosing an appropriate workflow to suit a particular desired state is not always obvious - indeed such manual workflows are conceptually similar to the imperative scripts which are no longer popular because of their unreliability.

Key Features

  • Automated composition of workflow
  • Enable unattended, autonomic, and distributed reconfiguration and auto-recovery
  • Achieve a desired state, and also preserve global constraints
  • A declarative configuration language: objects and procedures
  • Orchestrate or choreograph workflows
