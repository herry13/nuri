Nuri

  intelligent configuration management tool

"Just define your desired state and let Nuri achieves it for you."

✓ Object-oriented declarative configuration language.


✓ Automated composition of workflow.


✓ Orchestrating or choreographing configuration changes.


✓ Unattended, autonomic, and self-healing.


To install



$
sudo gem install nuri

Start local Nuri agent as cloud proxy



$
nuri console

nuri@user>
agent start

Model your desired state



nuri@user>
edit
